## 三、原因分析

### 1️⃣ 2月GMV最低（R$844,179）—— 季节性假期冲击
- **巴西狂欢节（Carnaval）**：2月是巴西传统狂欢节月份，大量消费者外出度假，购物需求下降
- **订单量低迷**：仅6,694笔，环比1月下降7.3%
- **客单价最低**：R$126.11，消费者购买意愿降低，倾向于低价商品
- **取消率最高**：达0.98%（上半年平均0.42%），假期期间订单取消概率增加
- **配送最慢**：平均16.84天，物流受假期影响明显延迟

### 2️⃣ 6月GMV第二低（R$865,124）—— 订单量骤降
- **订单量触底**：仅6,160笔，为上半年最低，环比5月下降10.1%
- **客单价不低**：R$140.44，说明不是价格问题，而是流量/需求减少
- **配送最快**：仅9.04天，物流效率提升但未能带动订单增长
- **可能原因**：6月为巴西冬季开始（南半球），部分品类进入淡季；或营销活动不足

### 3️⃣ 4-5月GMV最高（R$~996,600）—— 春季消费旺季
- **客单价攀升**：4月R$143.73 → 5月R$145.41，消费者购买力增强
- **配送效率提升**：从1-2月的14-17天缩短至11天，物流体验改善促进转化
- **订单量稳定**：维持在6,850-6,950笔区间，供需两旺
- **取消率低**：仅0.23%-0.34%，消费意愿强烈

### 4️⃣ 3月GMV排第三（R$983,213）—— 订单量最高
- **订单量7,188笔**：为上半年最高，但客单价R$136.79低于4-5月
- **配送天数15.97天**：仍受2月假期余波影响，物流恢复中

## 四、趋势总结

```
GMV趋势： 1月 → 2月 ↘ 3月 ↗ 4月 → 5月 → 6月 ↘
          (95万) (84万) (98万) (99.7万) (99.7万) (86.5万)
```

**整体呈现"假期疲软 → 春季回升 → 年中回落"的周期性波动：**
- **Q1（1-3月）**：受新年和狂欢节影响，GMV波动较大，2月为绝对低谷
- **Q2（4-6月）**：4-5月达到峰值，6月出现明显回落

---

## 五、业务建议

1. **2月淡季应对**：提前在1月底策划"狂欢节前大促"，锁定消费者节前购物需求；优化假期物流预案，缩短配送时长
2. **6月下滑应对**：6月订单量骤降但客单价不低，应加大营销引流力度，推出冬季品类专场（如保暖家居、电子娱乐等）
3. **4-5月旺季放大**：在客单价和配送效率双优的窗口期，加大广告投放和品类拓展，冲刺上半年业绩
4. **全年节奏参考**：建议将年度营销预算向3-5月倾斜，1-2月以维稳为主，6月需额外投入拉新促活
